FREQUENTLY ASKED QUESTIONS

Why am I receiving a "Nothing found to display" message after doing a search?

  • This message will display if there are no results to display which match the criteria entered or if you have not entered any criteria to search on. Only cases which were filed after July 31, 2009 will be returned in the search results. Please go to http://www.courts.state.va.us/ to locate any cases filed prior to July 31, 2009.
How can I search by case number?
  • Case number format includes a two character court type prefix followed by eight digits and a hyphen (-) with a two digit suffix. If the case number you are searching does not contain eight digits before the hyphen, try typing two zeroes after the first two digits. For example, if your case number is CL09-1234, try CL09001234-00. Please note that the two digit suffix does not indicate charge count. The suffix is used when subsequent filings are made under the base case such as Reinstatements.
How can I search by date in the Basic Search?
  • To search by a date in the Basic Search, the date must be in the following format: YYYYMMDD. For example, 20091231 would search for date December 31, 2009.

HELPFUL TIPS

Basic Search

  • The Basic Search is a full text search across all indexed fields.

    Example: If a user types John Smith into the Basic Search it will find all instances of John Smith located in all fields indexed. The user would also get instances of Smith John that are located in all fields indexed.

Advanced Search

  • The Advanced Search is a full text field-level search that can be accessed from the Advanced Search hyperlink on the main search page.

    Example: If a user types John Smith into the name field the user would get results for records that contained John Smith and Smith John only in the name field.

Results Sorting

  • ROAM utilizes client side sorting table to display results. A user can sort the results in the current page by double clicking on any of the column headers. Please note that this sorts only the results within the current page.
Wild Card Searching

  • ROAM supports Wildcard searching. A user can perform a Wildcard search by entering a ‘*’ after a minimum of 2 characters in order to utilize wildcard searching capability.

    Example: Entering Jo* Smith in ROAM will return results matching John Smith, Smith John, Joe Smith, Smith Joe, Joseph Smith, Smith Joseph, etc. Entering J* Smith into ROAM will only return results matching J Smith and Smith J. You must enter a minimum of 2 characters to use ROAM’s Wildcard searching capability

Faceting (Filter)

  • ROAM provides the ability to filter searches utilizing ROAM’s faceting functionality. The facet is located on the left hand side of your search results screen. When selecting an item in the facet it will filter your results to include only results that match that item. Selected filters are shown at the top of the facet. Users have the option of removing filters by clicking on them again in the top of the facet.
Exact Searches

  • Users can run an exact search by placing quotation marks (“ “) around the string of text they want to search on.

    Example: Entering “Smith John” will return results matching Smith John and will not return results matching John Smith

Results Per Page

  • The results per page in ROAM can be adjusted by the user to any of the allowed settings in the results per page drop down located in the lower right hand corner of the web page.
Export Options

  • ROAM allows the user to export the contents of the page into excel, pdf, and word using the Export feature located at the bottom of the search results
